# Simple Timer

A clean, responsive, and easy-to-use web-based countdown timer. Built to be simple and accessible, this application allows users to set a specific duration in hours, minutes, and seconds, and visually track the time remaining. 

## Features

- **Set Custom Times:** Input fields specifically for hours, minutes, and seconds.
- **Pause & Resume:** Easily stop the timer mid-countdown and resume from exactly where you left off.
- **Responsive Design:** Fully scaled to work seamlessly on modern desktop browsers as well as mobile devices.
- **Clean Interface:** Features a minimalist grid-pattern background and modern typography for a distraction-free experience.
- **Zero Dependencies:** Built entirely using fundamental web technologies without the bloat of heavy frameworks.

## Technologies Used

- **HTML5:** For the structural layout of the timer interface.
- **CSS3:** For styling, custom fonts (Google Montserrat), and fully responsive mobile media queries.
- **JavaScript (Vanilla):** For all core countdown logic, input parsing, time formatting, and dynamic DOM updates.
